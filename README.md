# 在线考试系统项目说明文档

## 1. 系统概述

在线考试系统是一个基于Spring Boot 3和Vue 3开发的全栈Web应用，旨在提供完整的在线考试解决方案。系统支持多角色用户（管理员、教师、学生）管理，实现了考试创建、题库管理、在线答题、自动评分等核心功能。

## 2. 技术架构

### 2.1 后端技术栈

- **核心框架**：Spring Boot 3.4.0
- **编程语言**：Kotlin 1.9.25
- **数据访问**：Spring Data JDBC
- **安全认证**：JWT (JSON Web Token)
- **数据库**：MySQL
- **文档导出**：Apache POI (Excel导出)

### 2.2 前端技术栈

- **核心框架**：Vue 3.5.13 (使用Composition API)
- **构建工具**：Vite 6.0.1
- **UI组件库**：Element Plus 2.9.0
- **路由管理**：Vue Router 4.5.0
- **HTTP客户端**：Axios 1.7.8
- **状态管理**：Vuex 4.0.2
- **工具库**：dayjs (日期处理)

### 2.3 系统架构

系统采用前后端分离架构：
- 后端提供RESTful API接口
- 前端通过Axios调用API实现数据交互
- JWT实现用户认证和授权
- 基于角色的访问控制(RBAC)实现权限管理

## 3. 数据模型

系统主要包含以下数据实体：

### 3.1 用户(User)
- 用户ID、用户名、密码、角色(管理员/教师/学生)、邮箱

### 3.2 课程(Course)
- 课程ID、课程名称、教师ID

### 3.3 题目(Question)
- 题目ID、内容、类型(单选/多选/判断/填空/简答)、选项、答案、难度、分类、创建时间、创建者ID

### 3.4 考试(Exam)
- 考试ID、标题、描述、开始时间、结束时间、创建者ID、创建时间、总分、题目列表、课程ID

### 3.5 考试提交(ExamSubmission)
- 提交ID、用户ID、考试ID、答案(JSON)、得分、提交详情、提交时间

### 3.6 考试题目关联(ExamQuestions)
- 考试ID、题目ID

### 3.7 课程选择(CourseSelection)
- 课程ID、学生ID

## 4. 功能模块

### 4.1 用户管理
- 用户注册、登录、退出
- 用户信息管理

### 4.2 课程管理

### 4.4 考试管理

- 考试列表展示

- 考试成绩统计与分析
- 成绩导出(Excel)

## 5. 接口设计

系统API接口遵循RESTful设计规范，主要包括：

### 5.1 用户接口
- `/api/users`：用户管理
- `/api/users/register`：用户注册
- `/api/users/login`：用户登录

### 5.2 课程接口
- `/api/courses`：课程管理
- `/api/course-selection`：选课管理

### 5.3 题目接口
- `/api/questions`：题目管理
- `/api/questions/categories`：题目分类

### 5.4 考试接口
- `/api/exams`：考试管理
- `/api/exams/{id}/details`：考试详情
- `/api/exams/random`：随机组卷

### 5.5 考试提交接口
- `/api/exam-submissions/submit`：提交考试
- `/api/exam-submissions/export`：导出成绩

## 8. 部署说明

### 8.1 环境要求
- JDK 21+
- MySQL 8.0+
- Node.js 16+
- npm 8+

### 8.2 后端部署
1. 创建MySQL数据库并导入`online_exam_system.sql`
2. 配置`application.properties`中的数据库连接信息
3. 使用Gradle构建项目：`./gradlew build`
4. 运行生成的JAR文件：`java -jar OnlineExamSystem-1.00.jar`

### 8.3 前端部署
1. 安装依赖：`npm install`
2. 开发环境运行：`npm run dev`
3. 生产环境构建：`npm run build`
4. 部署dist目录到Web服务器

## 9. 系统特点

1. **完整的考试流程**：从题库管理、试卷组建到在线答题、自动评分的完整流程
2. **多角色支持**：管理员、教师、学生三种角色，权限分明
3. **灵活的试卷组建**：支持手动选题和随机组卷两种方式
4. **多种题型支持**：单选题、多选题、判断题、填空题、简答题
5. **实时考试监控**：考试倒计时、自动提交功能
6. **数据导出功能**：支持成绩Excel导出
7. **响应式设计**：适配不同设备屏幕
